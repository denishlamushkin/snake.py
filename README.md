# snake.py
Snake game in [python](https://www.python.org/) with [pygame](https://github.com/pygame/pygame) library.
## controls
| Button | Description |
| :-: | :-: |
|Up|Move up|
|Down|Move down|
|Left|Move left|
|Right|Move right|
